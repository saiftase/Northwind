 - - -

 Use the following commands in the terminal to setup, test, and run Northwind.

###Setup

Installing Dependencies:
```bash
bower install angular
npm install
```

###Testing

Run Tests:
```bash
npm test
```
###Usage

Seed Database:
```bash
npm run seed
```

Start Server:
```bash
npm start
```

To view application, navigate to [http://localhost:3000/](http://localhost:3000/)
